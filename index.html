Comment lire les plans

Cherche la légende des murs — tableau avec des lettres (Type A, B, C...) qui décrit la composition de chaque mur. Info disponible dans la légende, les détails de coupe, ou le door schedule.

La porte est presque toujours 1¾" d'épaisseur. C'est le cadre qui change selon le mur.

Gypse ½" vs ⅝"

½" régulier — salles, chambres courant.

⅝" Type X — coupe-feu obligatoire : corridors, cages d'escalier, séparations entre logements.

Le jeu de ⅛" Manoir

Toujours ajouter ⅛" à l'épaisseur totale du mur fini pour le jeu de pose.

Cadrage fenêtres

Barre standard = 14' (168"). Les chutes sont réutilisées d'une fenêtre à l'autre — toujours entrer toutes les fenêtres ensemble pour un résultat optimal.
